Web SQL Database                    From Wikipedia, the free encyclopedia                            Jump to:     navigation,      search          Web SQL Database is a web page API for storing data in databases that can be queried using a variant of SQL.[1] The API is supported by Google Chrome,[2] Opera,[3] Safari[3] and the Android Browser. The W3C Web Applications Working Group ceased working on the specification in November 2010, citing a lack of independent implementations (i.e., the use of a database system other than SQLite as the backend) as the reason the specification could not move forward to become a W3C Recommendation.[1] One potential alternative storage standard is IndexedDB.
