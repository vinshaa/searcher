Please help improve this article by adding citations to reliable sources. Unsourced material may be challenged and removed. (August 2014) mod_parrot is an optional module for the Apache web server. It embeds a Parrot virtual machine interpreter into the Apache server and provides access to the Apache API to allow handlers to be written in Parrot assembly language, or any high-level language targeted to Parrot.
