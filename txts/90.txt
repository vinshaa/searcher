Do not fill this in!       ==Web Servers that implement FastCGI== :''Note: unless stated, completeness of FastCGI implementation is unknown'' * [[Apache HTTP Server]] ''(partial)'' ** Implemented by <tt>[http://httpd.apache.org/mod_fcgid/ mod_fcgid]</tt>. This module used to be third-party, but was granted to the [[Apache Software Foundation|ASF]] as an Apache HTTP Server subproject in 2009, shepherded by Chris Darroch.<ref>[http://httpd.apache.org/mod_fcgid/ FastCGI apache module mod_fcgid]</ref> It only supports [[UNIX socket]]s, no TCP sockets.<ref>[http://bugs.debian.org/cgi-bin/bugreport.cgi?bug=450748 Debian bug #450748: Please add support for TCP/IP FastCGI servers]</ref> ** A third-party module <tt>mod_fastcgi</tt> is also being used. For a while, this module no longer compiled properly under Apache 2.4.x,<ref>[http://web.archive.org/web/20120327033316/http://danielhe.com/2012/03/issues-with-apache-2-4-and-php-fpm/ Issues with Apache 2.4 and PHP-FPM]</ref> although that problem has been solved with a fork of the original project.<ref>[https://github.com/ByteInternet/libapache-mod-fastcgi/ libapache-mod-fastcgi on Github]</ref> ** Multiplexing of requests through a single connection is prohibited by Apache 1.x design,<ref>[http://cryp.to/publications/fastcgi/#AEN147 FastCGI ñ The Forgotten Treasure/ Section 2.3.]</ref> so this isn't supported ** In Apache 2.4, <tt>[http://httpd.apache.org/docs/2.4/mod/mod_proxy_fcgi.html mod_proxy_fcgi]</tt> was added, supporting TCP FastCGI servers. * [[Cherokee (web server)|Cherokee]]<ref>[http://www.cherokee-project.com/doc/modules_handlers_fcgi.html FastCGI for Cherokee]</ref> * [[Hiawatha (web server)|Hiawatha]]<ref>[http://www.hiawatha-webserver.org/howto/cgi_and_fastcgi FastCGI HOWTO for Hiawatha]</ref> ** Loadbalancing FastCGI support ** Supports chrooted FastCGI servers * [[Lighttpd]]<ref>[http://trac.lighttpd.net/trac/wiki/Docs:ModFastCGI FastCGI for Lighttpd]</ref> * [[LiteSpeed Web Server]] * [[Microsoft IIS]]<ref>{{cite web |url=http://www.iis.net/fastcgi |title=FastCGI Extension for IIS6.0 -
